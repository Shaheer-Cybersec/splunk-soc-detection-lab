# Splunk SOC Detection Engineering Lab

This project demonstrates a Security Operations Center (SOC) monitoring environment using Splunk Enterprise. The lab simulates cyber attacks from an attacker machine and detects them using centralized log monitoring.

## Lab Architecture

Machines used in the environment:

- Kali Linux (Attacker)
- Windows 10 (Endpoint)
- Metasploitable (Linux Target)
- Ubuntu Server (Splunk SIEM)

## Skills Demonstrated

- SIEM deployment
- Log ingestion
- Detection engineering
- Attack simulation
- Incident investigation

## Project Structure

- `architecture/` → Network diagrams and lab architecture  
- `labs/` → Step-by-step lab setup and execution  
- `detections/` → Detection rules and dashboards  
- `investigations/` → Incident case studies  
- `screenshots/` → Screenshots of lab setups, dashboards, and results
