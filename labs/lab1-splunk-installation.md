# Lab 1 — Splunk SIEM Installation

## Objective
Install and configure Splunk Enterprise SIEM on Ubuntu for centralized log monitoring.

## System
Ubuntu Server VM
- IP: 192.168.221.129
- RAM: 8GB
- CPU: 4 cores
- Disk: 100GB

## Installation Steps
1. Download Splunk DEB package
2. Install using `dpkg -i`
3. Start Splunk service
4. Enable auto-start
5. Access dashboard via `http://192.168.221.129:8000`

## Result
Splunk dashboard successfully running.
